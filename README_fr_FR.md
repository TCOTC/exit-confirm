Raccourcis de sortie :

- Windows : <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS : <kbd>Command</kbd> + <kbd>Q</kbd>

Note : L'auteur n'étant pas sûr du raccourci de sortie pour Linux, Linux n'est pas pris en charge.

Interaction avec la boîte de dialogue :

- Annuler la sortie : Appuyer sur <kbd>Esc</kbd> / Cliquer sur le bouton d'annulation
- Confirmer la sortie : Appuyer à nouveau sur le raccourci de sortie / Cliquer sur le bouton de sortie
