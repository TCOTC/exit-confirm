Atajos de salida:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Nota: Como el autor no está seguro del atajo de salida para Linux, Linux no está soportado.

Interacción del diálogo:

- Cancelar salida: Presionar <kbd>Esc</kbd> / Hacer clic en el botón de cancelar
- Confirmar salida: Presionar el atajo de salida nuevamente / Hacer clic en el botón de salir
