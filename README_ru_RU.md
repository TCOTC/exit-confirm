Горячие клавиши выхода:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Примечание: Поскольку автор не уверен в горячей клавише выхода для Linux, Linux не поддерживается.

Взаимодействие с диалоговым окном:

- Отменить выход: Нажмите <kbd>Esc</kbd> / Нажмите кнопку "Отмена"
- Подтвердить выход: Нажмите горячую клавишу выхода снова / Нажмите кнопку "Выход"
