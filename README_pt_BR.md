Atalhos de saída:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Nota: Como o autor não tem certeza sobre o atalho de saída para Linux, o Linux não é suportado.

Interação com a caixa de diálogo:

- Cancelar saída: Pressione <kbd>Esc</kbd> / Clique no botão Cancelar
- Confirmar saída: Pressione o atalho de saída novamente / Clique no botão Sair
