Scorciatoie di uscita:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Nota: Poiché l'autore non è sicuro della scorciatoia di uscita per Linux, Linux non è supportato.

Interazione con la finestra di dialogo:

- Annulla uscita: Premi <kbd>Esc</kbd> / Clicca sul pulsante Annulla
- Conferma uscita: Premi nuovamente la scorciatoia di uscita / Clicca sul pulsante Esci
