اختصارات الخروج:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

ملاحظة: نظرًا لأن المؤلف غير متأكد من اختصار الخروج لنظام Linux، فإن Linux غير مدعوم.

تفاعل النافذة المنبثقة:

- إلغاء الخروج: اضغط <kbd>Esc</kbd> / انقر على زر الإلغاء
- تأكيد الخروج: اضغط اختصار الخروج مرة أخرى / انقر على زر الخروج
