Çıkış kısayolları:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Not: Yazar Linux için çıkış kısayolundan emin olmadığı için Linux desteklenmemektedir.

İletişim kutusu etkileşimi:

- Çıkışı iptal et: <kbd>Esc</kbd> tuşuna basın / İptal düğmesine tıklayın
- Çıkışı onayla: Çıkış kısayoluna tekrar basın / Çıkış düğmesine tıklayın
