Beendigungstasten:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Hinweis: Da der Autor sich nicht sicher ist, welche Beendigungstaste für Linux verwendet wird, wird Linux nicht unterstützt.

Dialog-Interaktion:

- Beendigung abbrechen: <kbd>Esc</kbd> drücken / Auf die Schaltfläche "Abbrechen" klicken
- Beendigung bestätigen: Beendigungstaste erneut drücken / Auf die Schaltfläche "Beenden" klicken
