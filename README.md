Exit Shortcuts:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

p.s. Since the author is not sure about the exit shortcut for Linux, Linux is not supported.

Dialog interaction:

- Cancel exit: Press <kbd>Esc</kbd> / Click the cancel button
- Confirm exit: Press the exit shortcut again / Click the exit button
