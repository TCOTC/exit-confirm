Skróty wyjścia:

- Windows: <kbd>Alt</kbd> + <kbd>F4</kbd>
- macOS: <kbd>Command</kbd> + <kbd>Q</kbd>

Uwaga: Ponieważ autor nie jest pewien skrótu wyjścia dla Linuxa, Linux nie jest obsługiwany.

Interakcja z oknem dialogowym:

- Anuluj wyjście: Naciśnij <kbd>Esc</kbd> / Kliknij przycisk Anuluj
- Potwierdź wyjście: Naciśnij ponownie skrót wyjścia / Kliknij przycisk Zamknij
